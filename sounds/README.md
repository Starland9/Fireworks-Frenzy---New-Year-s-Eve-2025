Place your MP3 audio files in this folder with the following names:

## Required Files

| Filename | Description | Usage |
|----------|-------------|-------|
| `music.mp3` | **Background music** | Loops continuously during gameplay at lower volume (15%) |
| `pop.mp3` | Pop sound effect | When popping a regular firework |
| `explosion.mp3` | Explosion sound | When fireworks explode |
| `combo.mp3` | Combo sound | When achieving combo milestones (5x, 10x, etc.) |
| `bomb.mp3` | Bomb explosion | When hitting a bomb or game over |
| `golden.mp3` | Golden firework | When popping golden fireworks or collecting power-ups |

## Volume Levels

- **Background music**: 15% volume (quiet, ambient)
- **Sound effects**: 70-90% volume (loud, impactful)

## Notes

- All files should be in MP3 format
- If a sound file is missing, the game will use synthesized sounds as fallback
- Background music (`music.mp3`) will loop continuously once loaded
